# <image src="Images/main_logo.png" style="width: 2em; height: 2em; vertical-align: middle;" /> DrawGo

## 🚀 功能

- 随处可画
- 撤销和重做
- 隐藏画布，操作画布
- 画板选择
- 截图（未来支持）
- AI笔记（未来支持）
- 更多笔刷（未来支持）

[Releases 页面](https://github.com/largeprob/DrawGo/releases/latest) 下载最新 APK.

## ✨ 使用技巧
- 进入app后点击`开始`
- 长按并拖动来移动工具栏。
- 工具栏空闲 3 秒后变为 50% 透明。
- 启用 `隐藏画布时清空` 来在隐藏画布时将其清空。<br>
提示：触摸透传也会被自动禁用。
- 禁用 `启动时显示画布` 来在软件启动时默认隐藏画布。
